# ml-applications

Applications of machine learning concepts, tools, and techniques.

## Overview

A collection of hands-on projects and exercises covering core ML topics — from classical algorithms to deep learning. Each folder contains working code, notebooks, and notes.

## Projects

| [00-tools](00-tools/) | NumPy, Pandas, Linear Algebra | NumPy, Pandas |

## Setup
```bash
git clone https://github.com/garisonz/ml-applications.git
cd ml-applications
pip install -r requirements.txt
```
